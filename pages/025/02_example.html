Status Messages

Text color can convey meaning. Here are some typical examples:

Success!

Your action was completed successfully. ( text-green-700 on bg-green-50 )

Warning

Please check your input. Something might need attention. ( text-yellow-700 on bg-yellow-50 )

Error

An error occurred. Please try again. ( text-red-700 on bg-red-50 )

This is light text ( text-gray-100 ) on a dark background ( bg-gray-700 ), demonstrating good contrast for dark mode scenarios.
