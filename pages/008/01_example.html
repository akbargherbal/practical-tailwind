Example 1: `w-full` and `h-full` with a Defined Parent Height

The parent container below has a fixed height (`h-64`). The child element uses `w-full` and `h-full` to fill the parent completely.

Parent (w-3/4, h-64)

Child (w-full, h-full)

Key Classes Used:

  • w-full : Makes an element take up 100% of its parent's width.
  • h-full : Makes an element take up 100% of its parent's height, IF the parent has a defined height.
  • h-64 : An arbitrary height utility to define the parent's height.
