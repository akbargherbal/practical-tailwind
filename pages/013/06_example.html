Basic Outlined Button

A simple outlined button with a hover effect to change background and text color.

Key classes used:

  • border : Adds a 1px solid border.
  • border-blue-500 : Sets the border color.
  • text-blue-500 : Sets the text color.
  • py-2 px-4 : Adds padding.
  • rounded : Applies rounded corners.
  • hover:bg-blue-500 : Changes background color on hover.
  • hover:text-white : Changes text color on hover.
  • transition-colors duration-150 : Smooths the color transition.

This button has a transparent background by default.
