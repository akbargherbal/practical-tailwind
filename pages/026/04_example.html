Gotcha: Font Weight Availability

The key thing to remember is that Tailwind's font-light , font-bold , etc., classes will only have a visual effect if the font family you're currently using actually *includes* those specific weights.

For instance, if you're using a custom font that was only imported with 'normal' and 'bold' weights:

  • Applying font-light might result in the text appearing as 'normal' weight, not light.
  • Applying font-extrabold (if the font doesn't have it) might fall back to 'bold' or 'normal'.

The browser tries its best to render the requested weight. If unavailable, it typically falls back to the nearest available weight or the default (normal) weight. Always verify your font files or import links include the weights you intend to use with Tailwind's classes.
