Font Weight Utilities Showcase

font-light

This text is rendered with `font-light`. It appears thinner than the normal weight, suitable for de-emphasis or delicate text.

font-normal

This text is rendered with `font-normal`. This is typically the default weight for body text, offering good readability.

font-semibold

This text is rendered with `font-semibold`. It's bolder than normal but usually not as heavy as `font-bold`, good for subheadings.

font-bold

This text is rendered with `font-bold`. It provides strong emphasis, commonly used for headings or important statements.

Key Classes Used:

  • font-light - For lighter text.
  • font-normal - For standard weight text.
  • font-semibold - For semi-bold text.
  • font-bold - For bold text.

These classes control the thickness of text characters, allowing for varied emphasis and visual hierarchy.
