Basic Border Examples

Default Border

border

A simple, default 1px solid gray border.

Colored Border

border border-blue-500

Border color can be set using border-{color}-{shade} .

Thicker Border

border-4 border-green-500

Border width can be set using border-{width} (e.g., border-2 , border-4 , border-8 ).

Border with Rounded Corners

border-2 border-purple-600 rounded-lg

Borders work well with other utilities like rounded-lg .
