Example 1: Basic Success Alert

Success! Your profile has been updated.

Key Classes Used:

  • bg-green-100 : Sets a light green background.
  • border : Applies a default 1px solid border.
  • border-green-400 : Sets the border color to a medium green.
  • text-green-700 : Sets the text color to a dark green, ensuring good contrast.
  • px-4 : Adds horizontal padding.
  • py-3 : Adds vertical padding.
  • rounded : Applies rounded corners.
  • relative : (Optional, but often useful for positioning icons or close buttons).
