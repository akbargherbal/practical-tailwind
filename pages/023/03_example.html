Example 3: Informational Alert

Information

A new version of the software is available for download.

Key Classes Used (Variation):

  • bg-blue-100 : Sets a light blue background.
  • border-l-4 : Applies a thicker 4px left border.
  • border-blue-500 : Sets the left border color to a medium-dark blue.
  • text-blue-700 : Sets the text color to a dark blue, ensuring good contrast against the light background.
  • p-4 : Adds padding on all sides.
  • flex items-center : Used here to align an optional icon with the text.

This alert demonstrates a common alternative styling with a prominent left border, often used for info or warning messages. The principle of ensuring text readability through sufficient color contrast ( text-blue-700 on bg-blue-100 ) remains paramount.
