Success Alert Example

Success! Your profile has been updated.

Key Classes Used:

  • bg-green-100 : Sets a light green background.
  • border : Applies a default 1px solid border.
  • border-green-400 : Colors the border with a medium green.
  • text-green-700 : Sets a dark green text color for contrast.
  • px-4 : Adds horizontal padding.
  • py-3 : Adds vertical padding.
  • rounded : Applies rounded corners.
  • relative : (Optional, often used if you want to add absolutely positioned elements inside like a close button).
