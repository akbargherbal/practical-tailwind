Basic Clickable Button

This button uses the following key Tailwind classes for its appearance:

  • bg-blue-500 (background color)
  • text-white (text color)
  • py-2 (vertical padding)
  • px-4 (horizontal padding)

We've also added rounded for softer corners and hover:bg-blue-600 for interaction feedback, which are common for a "basic button look". Focus styles are also good practice for accessibility.
