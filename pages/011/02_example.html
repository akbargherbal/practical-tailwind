Button Styling: Gotcha and Best Practices

Gotcha: Using a <div> as a Button

It's common to see <div> elements styled to look like buttons. However, this approach has significant accessibility and usability drawbacks.

(Div) Looks like a Button

This <div> above is styled with bg-blue-500 text-white py-2 px-4 rounded . While visually similar, it is not keyboard focusable by default, and screen readers will not announce it as an interactive "button" element. This makes it inaccessible to many users.
