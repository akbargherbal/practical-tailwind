Tailwind CSS Shadow Examples

Small Shadow (`shadow-sm`)

This element has a subtle, small shadow. Good for elements that need a slight lift.

Medium Shadow (`shadow-md`)

This element uses the default medium shadow. A common choice for cards and prominent UI elements.

Large Shadow (`shadow-lg`)

This element features a larger, more pronounced shadow, making it stand out more.

Notice how each element has a bg-* class. This is important for shadows to render correctly, as explained in the "Gotcha" example.
