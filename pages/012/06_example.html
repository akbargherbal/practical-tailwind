Button Corner Styles

Using rounded-md

Using rounded-full

Explanation

The rounded-md class applies a medium border radius, giving the button slightly softened corners. This is a common style for a modern, clean look.

The rounded-full class applies a very large border radius, effectively making the button's ends fully circular, resulting in a "pill" or "capsule" shape. This is often used for call-to-action buttons or to make an element stand out.
