Gotcha: Inconsistent Padding

The "Gotcha" here is inconsistent padding, which can make the layout look messy. Notice the difference in internal spacing between the cards below.

Product A

Consistent padding (`p-4`) makes this card look balanced.

Price: $19.99

Product B (More Padding)

This card has `p-8`. If other cards use `p-4`, the internal spacing will look inconsistent.

Price: $29.99

Product C (Less Padding)

This card has `p-2`. The content feels more cramped compared to `p-4` or `p-8`.

Price: $9.99

Recommendation:

For a consistent UI, try to use the same padding values (e.g., `p-4` or `p-6`) for all similar card components unless a specific design requires variation.
