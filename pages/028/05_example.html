Text Alignment Examples

Text Left:

This paragraph uses the text-left class. The text content within this block-level paragraph element will align to the left. This is often the default behavior for most languages.

Text Center:

This paragraph uses the text-center class. All text content inside this paragraph will be centered relative to the paragraph's own boundaries.

Text Right:

This paragraph uses the text-right class. The text here will align to the right edge of this paragraph element.

Each of these paragraphs is a block-level element. The text-* utilities control the alignment of the text within these containers.
